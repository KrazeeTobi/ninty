ドンキーコング64（仮称）
1999 NINTENDO. Game by RARE.
1999年12月発売予定

●メモリー拡張パック専用　●振動パック対応
キャラクターはお馴染みのドンキーとディディーに加え新たにタイニー、チャンキー、ランキーの5人が登場。それぞれのキャラクターならではの特殊能力をフルに使って様々なステージで大暴れ。ひとつのカセットに5つのゲームが詰まってるかのような盛りだくさんなゲーム内容！



ディディーを樽から発射し、ドンキーをトロッコに乗せて突っ走り、タイニーを使ってステージを探索したりといった豊富なアクション！！

いろんなミニゲームや、隠しボーナスステージが盛りだくさん！！プイレヤーを絶対飽きさせないぞ！
※「ドンキーコング64」（仮）をプレイするには、
「メモリー拡張パック」が必要です。
たとえば・・・
いろいろなカラーのリアルタイムライティング効果が綺麗なテクスチャーや凝った作りの背景と相まって、驚きのビジュアルを実現しているのだ！
