• Code Name: S.T.E.A.M.

『Code Name: S.T.E.A.M.』は、『ファイアーエムブレム』や『ファミコンウォーズ』シリーズを手がけたインテリジェントシステムズの新作シミュレーション・アクションゲームです。スチームパンクな時代設定、アメリカンコミック調のグラフィックで、独特の世界観を味わうことができます。
プレイヤーはアメリカの16代大統領、リンカーンが率いる特殊部隊S.T.E.A.M.の個性豊かなエージェントたちから4人を選び、地球を侵略するエイリアンたちを撃退していきます。
ゲームはターン制でありながらも3Dサードパーソンビューのアクションで進行し、サードパーソンシューターの臨場感をシミュレーションゲームで味わうことができます。また、オンラインマルチプレイや、プレイヤー同士の対戦も楽しめます。
